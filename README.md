## 현재 진행상황
* 안드로이드와 파이어베이스 store연동   
* Login and SignUp 레이아웃과 기능 구현 완료   
* 회원가입된 정보를 토대로 로그인 가능   
* Navigation Drawer을 이용한 UI & UX 구조 개선
* Post 레이아웃과 기능 구현완료 
* 가입된 이메일을 통해 이메일 인증 구현완료
* 게시글 작성기능 구현완료
